Img2Sign(pending)
Personal project that would allow for easy import of images to game Starbound in form of game animated signs.
Project is not finished and still needs a lot of work until be production ready but core functionality is working:

  • impoting images
  • pixel drawing tool
  • assigning starbound sign specific valuse(border colors/in-game glow/sign wiring for logic)
  • frames for sign animations
  • layers for single animation frame
  • starbound commands generator(commands are printed in browser console)

modding tools

GMC - Gothic Mod Composer(in development)
Tool for building Gothic II Mod Projects with configurable workflow of building process.
More information will be available after release of The History of Khorinis.

GMC-TE- Gothic Mod Composer - Translation Extension(in development)
Tool that allows for extracting written text form Gothic II script files, translation of them to different languages with easy to use interface and placing translated texts back to game scripts.
More information will be available after release of The History of Khorinis.
